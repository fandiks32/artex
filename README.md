# Artex : Augmented Reality Text Translation in Android
Artex is a Android Application

## Requirements ##
* Eclipse
* Vuforia SDK 4.0
* Java
* Xml

## How to use ##
1. Export Project on Eclipse
2. Install Vuforia SDK on project(build path)
3. Happy Coding
